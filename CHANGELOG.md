# DocuFormat Pro 更新日志

本文档整合了 **DocuFormat Pro** 自动化 Word 排版工具的开发进度、已实现功能以及历次更新记录。

---

## 🚀 已实现核心功能

### 1. 页面设置 (Page Setup)
- [x] **边距调整**：支持精确设置上、下、左、右页边距（单位：厘米）。
- [x] **自动化应用**：一键将边距规则应用到文档的所有节（Section）。

### 2. 字体与文字格式 (Typography)
- [x] **中西文分离**：支持分别为中文字体和英文字体设置名称。
- [x] **字号映射**：支持中文字号（如"小四"）与磅值（12pt）的自动映射。
- [x] **文字装饰**：支持加粗（Bold）设置。
- [x] **自定义颜色**：支持通过 Hex 颜色码（如 #FF0000）自定义正文文字颜色，默认勾选启用。

### 3. 段落格式 (Paragraph)
- [x] **首行缩进**：支持按字符数设置首行缩进。
- [x] **行距控制**：支持单倍、1.5倍、双倍、多倍、固定值及最小值行距。
- [x] **段间距**：支持设置段前和段后间距（以"行"为单位）。

### 4. 标题样式 (Heading Styles)
- [x] **多级支持**：支持标题 1、标题 2、标题 3 的独立样式定义。
- [x] **简化布局**：紧凑表格式布局，每级标题一行，仅暴露中文字体（宋体/黑体/隶书/仿宋）和字号。
- [x] **固定参数**：英文字体固定 Times New Roman，段前/段后固定 6pt，行距固定 1.4 倍。
- [x] **独立颜色**：每个层级的标题均可独立开启和设置自定义字体颜色。
- [x] **样式强制生效**：处理时清除标题段落的直接格式覆盖，确保样式定义生效。

### 5. 表格重塑 (Table Formatting)
- [x] **可选启用**：通过"应用表格样式"开关控制是否格式化表格（默认不勾选）。
- [x] **自动适应**：表格自动调整为窗口宽度。
- [x] **对齐方式**：支持单元格内文字的水平与垂直对齐设置。
- [x] **边距与行距**：支持设置表格内段落的间距。
- [x] **表头管理**：支持首行加粗及跨页重复表头。
- [x] **智能同步**：可选项支持表格字体与正文保持一致。

### 6. 附加功能 (Extended Features)
- [x] **前置页插入**：支持自动从模板文档提取前 4 页（如封面、前言）并插入到目标文档开头。
- [x] **题注处理**：自动处理表格和图片的题注编号与格式（默认不勾选，可独立开关）。
- [x] **清除背景**：自动清除文档中的文字高亮和底纹背景。
- [x] **排版前确认**：点击"开始排版"后弹出确认窗口，列举即将应用的所有参数。

### 7. 模板管理 (Template Management)
- [ ] **当前状态**：模板管理区块暂时禁用，待功能测试完成后开放。
- [x] **格式提取**：支持"📥 从文档提取模板"，自动读取已有文档的排版参数。
- [x] **持久化存储**：支持保存、加载及删除自定义排版模板。
- [x] **默认恢复**：支持一键恢复到标准排版建议值。

## 📅 更新历史

### 2026-05-25

#### 新增功能
- **表格相关默认不勾选**：表格样式和题注处理开关默认关闭。
- **模板管理暂时禁用**：模板管理区块整体置灰，标注"暂未开放"，待测试完成后启用。
- **字体颜色默认勾选**：正文和标题的"设置字体颜色"开关默认启用。
- **标题左缩进清零**：标题段落强制清除左缩进和首行缩进，避免继承正文的 2 字符首行缩进。

#### Bug 修复
- **标题样式不生效**：修复了修改标题样式定义后，文档中标题段落因直接格式覆盖而不生效的问题。处理时会清除标题段落的直接字体格式并重新应用样式。

#### convert_to_word.py 脚本改进
- **标题使用 Word 内置样式**：`add_heading` 改为使用 `doc.add_paragraph(style='Heading N')`，确保 Word 导航窗格和目录能正确识别标题。
- **忽略 # 一级标题**：从 `##` 开始映射为 Heading 1，`###` 映射为 Heading 2，`####` 映射为 Heading 3。

### 2026-05-15

- **[新增] 排版前二次确认对话框**：
    - 点击"开始排版"后弹出确认窗口，列举所有即将应用的格式化参数。
    - 显示文件数量、页面设置、正文字体、标题样式、表格/题注处理状态、前置页等信息。
    - 用户确认后才开始处理，取消则返回等待状态。
- **[新增] 标题格式面板简化重构**：
    - 标题格式区块从 Expander 展开式改为紧凑表格式布局。
    - 每级标题一行，仅暴露中文字体（下拉：宋体/黑体/隶书/仿宋）和字号选择。
    - 英文字体固定 Times New Roman，段前/段后固定 6pt，行距固定 1.4 倍。

### 2026-05-14

- **[新增] 表格和题注可选处理**：
    - 为表格样式和题注处理添加了独立的开关选项。
    - 勾选"应用表格样式"可启用表格格式化（边框、对齐、行距等）。
    - 勾选"自动处理题注编号"可启用表格/图片题注的自动编号。
    - 不勾选时保留文档原有的表格和题注样式。
- **[修复] 文件重新处理状态重置**：
    - 修复了处理完一个文件后，重新选择同一文件时状态未被重置的问题。
    - 已完成或失败的文件重新添加时，状态现在正确重置为"排队中"。
- **[修复] 表格背景色混乱**：
    - 修复了不勾选"应用表格样式"时，表格单元格背景色被设置成混乱状态的问题。
    - 现在无论是否应用表格格式，都会正确清除表格单元格的背景色。
- **[修复] 多文件处理 COM 对象失效**：
    - 修复了处理第二个文件时出现"COM Object that has been separated from its underlying RCW"错误的问题。
    - 改进了 Word COM 对象的生命周期管理，每次处理都在独立的 STA 线程内创建和释放 Application 实例。
    - 增强了异常传递机制，确保处理失败能被正确标记而不是误判为成功。

### 2026-05-12

- **[修复] 转换闪退问题**：
    - 解决了同时勾选多个字体颜色设置时可能导致的后台线程崩溃。
    - 修复了加载旧版本模板时因缺少颜色字段导致的空引用异常。
    - 增强了十六进制颜色代码的解析容错性。
- **[优化] 异常捕获**：
    - 优化了 Word COM 接口的调用稳定性，处理失败时显示具体错误堆栈而不会直接闪退。
