# DocuFormat Pro 功能进度报告

本文档概述了 **DocuFormat Pro** 自动化 Word 排版工具的当前开发进度、已实现功能以及最近的修复项。

## 🚀 已实现核心功能

### 1. 页面设置 (Page Setup)
- [x] **边距调整**：支持精确设置上、下、左、右页边距（单位：厘米）。
- [x] **自动化应用**：一键将边距规则应用到文档的所有节（Section）。

### 2. 字体与文字格式 (Typography)
- [x] **中西文分离**：支持分别为中文字体和英文字体设置名称。
- [x] **字号映射**：支持中文字号（如“小四”）与磅值（12pt）的自动映射。
- [x] **文字装饰**：支持加粗（Bold）设置。
- [x] **自定义颜色**：支持通过 Hex 颜色码（如 #FF0000）自定义正文文字颜色。

### 3. 段落格式 (Paragraph)
- [x] **首行缩进**：支持按字符数设置首行缩进。
- [x] **行距控制**：支持单倍、1.5倍、双倍、多倍、固定值及最小值行距。
- [x] **段间距**：支持设置段前和段后间距（以“行”为单位）。

### 4. 标题样式 (Heading Styles)
- [x] **多级支持**：支持标题 1、标题 2、标题 3 的独立样式定义。
- [x] **对齐方式**：支持左对齐、居中、右对齐及两端对齐。
- [x] **独立颜色**：每个层级的标题均可独立开启和设置自定义字体颜色。

### 5. 表格重塑 (Table Formatting)
- [x] **自动适应**：表格自动调整为窗口宽度。
- [x] **对齐方式**：支持单元格内文字的水平与垂直对齐设置。
- [x] **边距与行距**：支持设置表格内段落的间距。
- [x] **表头管理**：支持首行加粗及跨页重复表头。
- [x] **智能同步**：可选项支持表格字体与正文保持一致。

### 6. 附加功能 (Extended Features)
- [x] **前置页插入**：支持自动从模板文档提取前 4 页（如封面、前言）并插入到目标文档开头。
- [x] **题注处理**：自动处理表格和图片的题注编号与格式。
- [x] **清除背景**：自动清除文档中的文字高亮和底纹背景。

### 7. 模板管理 (Template Management)
- [x] **格式提取**：支持“📥 从文档提取模板”，自动读取已有文档的排版参数。
- [x] **持久化存储**：支持保存、加载及删除自定义排版模板。
- [x] **默认恢复**：支持一键恢复到标准排版建议值。

## 🛠 最近修复与优化 (2026-05-14)

- **[新增] 表格和题注可选处理**：
    - 为表格样式和题注处理添加了独立的开关选项。
    - 勾选"应用表格样式"可启用表格格式化（边框、对齐、行距等）。
    - 勾选"自动处理题注编号"可启用表格/图片题注的自动编号。
    - 不勾选时保留文档原有的表格和题注样式。

- **[修复] 文件重新处理状态重置**：
    - 修复了处理完一个文件后，重新选择同一文件时状态未被重置的问题。
    - 已完成或失败的文件重新添加时，状态现在正确重置为"排队中"。

- **[修复] 表格背景色混乱**：
    - 修复了不勾选"应用表格样式"时，表格单元格背景色被设置成混乱状态的问题。
    - 现在无论是否应用表格格式，都会正确清除表格单元格的背景色。

- **[修复] 多文件处理 COM 对象失效**：
    - 修复了处理第二个文件时出现"COM Object that has been separated from its underlying RCW"错误的问题。
    - 改进了 Word COM 对象的生命周期管理，每次处理都在独立的 STA 线程内创建和释放 Application 实例。
    - 增强了异常传递机制，确保处理失败能被正确标记而不是误判为成功。
